# Ember & Oak — Wood-Fired Kitchen

A single-page restaurant website built with **HTML, CSS, and vanilla JavaScript**. Designed for a fictional wood-fired restaurant, featuring an interactive menu with a working cart/order system.

## Live Demo
[View it here]( https://abdul5550.github.io/Ember-Oak/) 
## Built With
- HTML5
- CSS3 (custom properties, animations, responsive grid layout)
- JavaScript (Vanilla — no frameworks or libraries)
- Google Fonts (Fraunces, Manrope, IBM Plex Mono)

## Features
- **Interactive menu system** — dishes and drinks are rendered dynamically from a JS data array, with filter tabs (All / Meals / Drinks)
- **Live order cart** — add/remove items, running total calculation, slide-out cart drawer
- **Toast notifications** — confirms when an item is added to the order
- **Scroll-based effects** — sticky header that changes style on scroll, scroll-triggered fade-in animations using `IntersectionObserver`
- **Fully responsive design** — mobile navigation menu toggle, fluid typography with `clamp()`
- **Custom aesthetic** — warm dark theme with gold accents, subtle grain texture overlay, and animated logo flicker

## Sections
- Hero / landing
- About
- Menu (with live ordering)
- Visit / location
- Footer with contact info and social links

## What I Learned / Practiced
This project was built to practice structuring a full front-end site without a framework — including dynamic DOM rendering from JS data, state management for a cart system using plain JavaScript, and smooth scroll/reveal animations using the Intersection Observer API.

## Notes
This is a concept/practice project built to demonstrate front-end development skills — not a real restaurant. All content, menu items, and contact details are placeholder/fictional.
